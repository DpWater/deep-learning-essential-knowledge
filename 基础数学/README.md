## 推荐学习顺序

1. 高等数学：先把导数、积分、优化概念打牢。
2. 线性代数：重点理解“变换”和“特征空间”。
3. 概率与离散：建立建模与统计判断能力。
4. 回到 D2L 线性网络章节，做线性回归与 softmax 的最小项目实践。
