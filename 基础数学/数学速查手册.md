# 数学词典（深度学习超易版）

目标：你遇到一个数学概念时，能在 30 秒内知道“它是什么、长什么样、像什么物理过程、在深度学习哪里用、怎么立刻用”。

## 怎么用这本词典
- 每个词条都固定 6 项：`含义`、`定理/公式`、`图像意义`、`物理意义`、`深度学习运用`、`拿来就用`。
- 看不懂时先看 `含义` + `拿来就用`，再看公式。
- 记号统一：
  - `X`: 输入（`[batch, d_in]`）
  - `W`: 权重（`[d_in, d_out]`）
  - `y`: 标签，`p`: 预测概率
  - `L`: loss，`theta`: 参数

---

## 1. 向量与矩阵

### 词条 1：向量（Vector）
- 含义：一串有顺序的数字，可看成“一个点”或“一个方向”。
- 定理/公式：`x = [x1, x2, ..., xd]^T`。
- 图像意义：2D/3D 里是一根箭头。
- 物理意义：速度、力、位移都可以是向量。
- 深度学习运用：词向量、图像特征向量、用户 embedding。
- 拿来就用：模型中一条样本常写成 `x in R^d`。

### 词条 2：矩阵（Matrix）
- 含义：把很多向量排成表格，或把“线性变换”写成表格。
- 定理/公式：`X in R^{n x d}`，`n` 条样本，`d` 个特征。
- 图像意义：把空间做旋转、拉伸、压缩。
- 物理意义：多输入多输出系统的线性响应表。
- 深度学习运用：全连接层、注意力投影、批量数据。
- 拿来就用：先检查 shape，再写代码。

### 词条 3：内积（Dot Product）
- 含义：两个向量“方向有多一致”。
- 定理/公式：`x^T y = ||x|| ||y|| cos(theta)`。
- 图像意义：`x` 在 `y` 方向上的投影。
- 物理意义：功 `W = F * s * cos(theta)`。
- 深度学习运用：注意力分数 `q·k`，向量相似度检索。
- 拿来就用：做召回时，常用“归一化后内积”算相似度。

### 词条 4：范数（Norm）
- 含义：向量/矩阵“有多大”。
- 定理/公式：
  - `L1 = sum |xi|`
  - `L2 = sqrt(sum xi^2)`
  - `||A||_F = sqrt(sum Aij^2)`
- 图像意义：L2 单位球是圆；L1 单位球是菱形。
- 物理意义：L2 像“能量”，L1 像“总消耗”。
- 深度学习运用：L2 正则（weight decay）、梯度裁剪。
- 拿来就用：过拟合优先试 `weight_decay=1e-4`。

### 词条 5：矩阵乘法（Matrix Multiplication）
- 含义：把一个变换作用到数据上。
- 定理/公式：`Y = XW + b`，其中 `X:[n,d_in]`，`W:[d_in,d_out]`。
- 图像意义：先旋转再缩放再平移（加偏置）。
- 物理意义：信号经过一个线性系统。
- 深度学习运用：几乎所有线性层和注意力投影。
- 拿来就用：`mat1 and mat2 shapes cannot be multiplied` 时先对齐中间维。

### 词条 6：SVD（奇异值分解）
- 含义：任意矩阵都能拆成“旋转 + 拉伸 + 旋转”。
- 定理/公式：`A = U Sigma V^T`。
- 图像意义：圆被变换成椭圆，椭圆半轴就是奇异值。
- 物理意义：系统对不同方向的增益强弱。
- 深度学习运用：低秩压缩、LoRA 思路、权重分析。
- 拿来就用：线性层太大时，先尝试低秩近似压缩。

## 2. 微积分与反向传播

### 词条 7：导数与梯度（Gradient）
- 含义：函数在当前位置“往哪走升得最快”。
- 定理/公式：`grad f = [df/dx1, ..., df/dxd]`。
- 图像意义：等高线图里，梯度垂直等高线。
- 物理意义：势场中的受力方向。
- 深度学习运用：参数更新方向来自 `-grad`。
- 拿来就用：优化时永远沿负梯度走：`theta = theta - lr * grad`。

### 词条 8：链式法则（Chain Rule）
- 含义：复合函数求导要“层层相乘”。
- 定理/公式：`dL/dx = dL/du * du/dx`。
- 图像意义：每一层都把梯度缩放一次。
- 物理意义：串联系统总增益是各级增益乘积。
- 深度学习运用：反向传播的核心。
- 拿来就用：排查梯度消失时，从后往前看每层导数是否过小。

### 词条 9：Jacobian（雅可比）
- 含义：向量函数的一阶导“总表”。
- 定理/公式：`Jij = dyi/dxj`。
- 图像意义：局部线性变换矩阵。
- 物理意义：小扰动输入如何传到输出。
- 深度学习运用：Flow 模型、敏感性分析。
- 拿来就用：要做鲁棒性分析时，看 Jacobian 范数。

### 词条 10：Hessian（海森）
- 含义：二阶导矩阵，告诉你曲率。
- 定理/公式：`Hij = d^2f/(dxi dxj)`。
- 图像意义：碗有多陡、哪个方向最陡。
- 物理意义：系统刚度矩阵。
- 深度学习运用：学习率稳定性、二阶优化近似。
- 拿来就用：loss 抖动大时先降学习率，本质是曲率太大。

### 词条 11：泰勒展开（Taylor）
- 含义：用低阶多项式近似函数局部行为。
- 定理/公式：`f(x+dx) ~= f(x) + g^T dx + 1/2 dx^T H dx`。
- 图像意义：用切线/抛物面贴近原函数。
- 物理意义：小范围线性化建模。
- 深度学习运用：收敛分析、步长选择。
- 拿来就用：大步更新前，先确认局部二次近似是否会发散。

## 3. 概率统计

### 词条 12：条件概率与贝叶斯
- 含义：看到新证据后，更新原先判断。
- 定理/公式：`P(A|B)=P(B|A)P(A)/P(B)`。
- 图像意义：在“满足 B 的区域”里重新算比例。
- 物理意义：观测后更新系统状态估计。
- 深度学习运用：后验推断、概率分类解释。
- 拿来就用：类别先验变了时，做后验校正。

### 词条 13：期望、方差、协方差
- 含义：
  - 期望：平均位置
  - 方差：波动大小
  - 协方差：两变量是否同涨同跌
- 定理/公式：
  - `Var(X)=E[X^2]-E[X]^2`
  - `Cov(X,Y)=E[(X-EX)(Y-EY)]`
- 图像意义：点云的中心和散布椭圆。
- 物理意义：信号均值和噪声强度。
- 深度学习运用：标准化、BN/LN、不确定性建模。
- 拿来就用：输入先标准化，通常收敛更快更稳。

### 词条 14：MLE（最大似然）
- 含义：选一组参数，让观测数据“最可能出现”。
- 定理/公式：`theta* = argmax sum log p(xi|theta)`。
- 图像意义：在参数空间找最高峰。
- 物理意义：找最能解释实验数据的模型参数。
- 深度学习运用：分类/回归/语言模型训练本质都可看成 MLE。
- 拿来就用：先写概率模型，再最小化负对数似然。

### 词条 15：MAP（最大后验）
- 含义：在 MLE 上加入先验偏好。
- 定理/公式：`argmax [log p(D|theta) + log p(theta)]`。
- 图像意义：在似然地形上叠加先验约束。
- 物理意义：数据证据与先验知识折中。
- 深度学习运用：L2 正则可看成高斯先验下的 MAP。
- 拿来就用：数据少时加正则，相当于给参数“拉回去”。

## 4. 信息论与损失函数

### 词条 16：熵（Entropy）
- 含义：不确定性有多大。
- 定理/公式：`H(P) = -sum p log p`。
- 图像意义：分布越平均，熵越高。
- 物理意义：系统无序度。
- 深度学习运用：不确定性估计、标签平滑。
- 拿来就用：模型过度自信时，可用 label smoothing。

### 词条 17：交叉熵（Cross Entropy）
- 含义：用模型分布 `Q` 描述真实分布 `P` 的代价。
- 定理/公式：`H(P,Q) = -sum p log q`。
- 图像意义：`Q` 和 `P` 越不一致，代价越高。
- 物理意义：编码失配导致额外码长。
- 深度学习运用：分类任务标准 loss。
- 拿来就用：PyTorch 多分类直接 `CrossEntropyLoss`（输入 logits）。

### 词条 18：KL 散度
- 含义：两个分布差了多少（不对称）。
- 定理/公式：`KL(P||Q)=sum p log(p/q) >= 0`。
- 图像意义：`Q` 拟合 `P` 的“信息损失”。
- 物理意义：错误模型带来的额外代价。
- 深度学习运用：VAE、知识蒸馏、分布对齐。
- 拿来就用：蒸馏时常用 `KL(student || teacher)` 变体。

### 词条 19：Softmax + CE 的关键梯度
- 含义：多分类最重要的一条梯度公式。
- 定理/公式：`dL/dz = p - y`（`y` 为 one-hot）。
- 图像意义：把预测概率向真实类别方向推。
- 物理意义：误差力场推动 logits 修正。
- 深度学习运用：所有分类头的反向传播。
- 拿来就用：不要手写 softmax+log，直接用框架融合实现。

### 词条 20：MSE / MAE / Huber
- 含义：回归常用损失三件套。
- 定理/公式：
  - `MSE = mean((y-yhat)^2)`
  - `MAE = mean(|y-yhat|)`
  - Huber：小误差像 MSE，大误差像 MAE
- 图像意义：MSE 是平滑抛物线，MAE 是尖角折线。
- 物理意义：MSE 更重罚大误差，MAE 更抗离群点。
- 深度学习运用：数值预测、重建模型。
- 拿来就用：数据离群点多时，优先试 Huber。

## 5. 优化与训练稳定性

### 词条 21：SGD / Momentum
- 含义：沿负梯度下降，动量让方向更平滑。
- 定理/公式：
  - `v = beta*v + (1-beta)*g`
  - `theta = theta - lr*v`
- 图像意义：峡谷地形里减少左右抖动。
- 物理意义：带惯性的阻尼运动。
- 深度学习运用：CV 中常见 `SGD+momentum`。
- 拿来就用：`momentum=0.9` 是常用默认值。

### 词条 22：Adam
- 含义：给每个参数自适应步长。
- 定理/公式：一阶矩 `m` + 二阶矩 `v` + 偏差修正。
- 图像意义：不同方向自动调节步长大小。
- 物理意义：各方向阻尼不同的运动系统。
- 深度学习运用：Transformer/LLM 常用。
- 拿来就用：先用 Adam 跑通，再考虑换 SGD 做泛化微调。

### 词条 23：学习率（Learning Rate）
- 含义：每次参数更新走多大一步。
- 定理/公式：`theta <- theta - lr * grad`。
- 图像意义：步子过大跨过谷底，过小则爬太慢。
- 物理意义：离散积分步长决定稳定性。
- 深度学习运用：warmup、cosine decay。
- 拿来就用：loss 发散先减小 lr；前几轮加 warmup。

### 词条 24：梯度裁剪（Gradient Clipping）
- 含义：限制梯度不要过大。
- 定理/公式：若 `||g||>c`，则 `g <- c*g/||g||`。
- 图像意义：把太长箭头截短。
- 物理意义：限制系统瞬时最大冲击力。
- 深度学习运用：RNN、长序列 Transformer、混合精度。
- 拿来就用：先试 `clip_grad_norm_=1.0`。

## 6. 数值稳定与归一化

### 词条 25：LogSumExp
- 含义：稳定算 `log(sum(exp(x)))`。
- 定理/公式：`m=max(x)`，`lse = m + log(sum(exp(x-m)))`。
- 图像意义：把曲线整体平移后再指数，避免爆炸。
- 物理意义：改变参考零点，不改变相对差。
- 深度学习运用：softmax、能量模型、CRF。
- 拿来就用：凡是 `log(sum(exp()))` 一律换 `logsumexp`。

### 词条 26：BCEWithLogits（数值稳定版 BCE）
- 含义：把 sigmoid 和 BCE 合在一起，避免数值问题。
- 定理/公式：内部使用稳定重写，避免 `log(0)`。
- 图像意义：同样曲线，但计算路径更稳。
- 物理意义：避免极端值导致计算溢出。
- 深度学习运用：二分类/多标签分类。
- 拿来就用：二分类不要先 sigmoid 再 BCE；直接用 `BCEWithLogitsLoss`。

### 词条 27：BatchNorm / LayerNorm
- 含义：把激活拉到可控尺度，再学回合适缩放和平移。
- 定理/公式：`xhat=(x-mu)/sqrt(var+eps)`，`y=gamma*xhat+beta`。
- 图像意义：先“校准量纲”，再“调音量”。
- 物理意义：让系统状态保持在稳定工作区。
- 深度学习运用：BN 常见于 CNN；LN 常见于 Transformer。
- 拿来就用：小 batch 或序列模型优先 LN。

### 词条 28：高斯分布与重参数化
- 含义：高斯描述噪声；重参数化让采样可反传。
- 定理/公式：`z = mu + sigma * eps`，`eps~N(0,I)`。
- 图像意义：先在标准正态采点，再平移缩放。
- 物理意义：随机性来自外部噪声，参数控制变换。
- 深度学习运用：VAE、扩散模型中的噪声过程。
- 拿来就用：有随机隐变量但要训练时，优先用重参数化写法。

## 7. 按任务快速定位
- 做多分类：词条 `17, 19, 25`。
- 做回归：词条 `20, 13`。
- 训练不稳定：词条 `10, 23, 24, 25, 27`。
- 模型过拟合：词条 `4, 15`。
- 向量检索/推荐：词条 `3, 4`。
- 模型压缩：词条 `6`。
- 做 VAE/生成：词条 `18, 28`。

## 8. 开训前 60 秒检查
- `shape`：`X @ W + b` 是否对齐。
- `loss`：任务和损失是否匹配。
- `logits`：分类损失是否直接吃 logits。
- `lr`：先小范围试跑，看 loss 是否单调下降。
- `grad_norm`：是否爆炸（必要时裁剪）。
- `normalize`：输入与中间层是否做了合理归一化。
