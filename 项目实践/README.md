## 基于你提供资料的专项实践（线性网络）

### 实践 1：线性回归最小工程

- 目标：用线性回归完成一个回归任务并输出 MAE/RMSE/R2。
- 数据：可先用 D2L 合成数据，再替换为你自己的 CSV 数据。
- 验收：
- 能跑通训练与验证。
- 能解释特征缩放对收敛速度的影响。

### 实践 2：softmax 多分类最小工程

- 目标：完成 `logits -> CrossEntropyLoss -> accuracy/F1` 全流程。
- 数据：Fashion-MNIST 或任意小型图像/表格分类数据。
- 验收：
- 标签与损失函数格式匹配正确。
- 可输出混淆矩阵并分析错误类别。

### 实践 3：数值稳定性排障演练

- 目标：构造并修复一次 `loss=nan` 问题。
- 检查项：学习率、输入异常值、损失实现方式。
- 验收：记录“问题现象 -> 定位步骤 -> 修复结果”。

### 实践 4：正则化与泛化对比实验

- 目标：比较无正则、L2、dropout 三组策略。
- 验收：
- 提供训练/验证曲线。
- 给出偏差-方差角度的结论。
