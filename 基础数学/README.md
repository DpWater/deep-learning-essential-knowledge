# 基础数学

用于整理深度学习必备数学知识，并与工程实践强绑定。

## 学习目标

- 从“会套公式”升级到“会理解、会实现、会排错”。
- 把数学知识直接映射到训练代码和模型行为解释。
- 形成可复用的学习闭环：前置检查 -> 学习 -> 练习 -> 验收。

## 推荐学习顺序

1. 01 向量、矩阵与张量
2. 02 线性变换、特征值与特征向量
3. 03 导数、梯度、Jacobian 与 Hessian
4. 04 概率分布与最大似然估计
5. 05 信息论基础（熵、交叉熵、KL 散度）
6. 06 数值稳定性与误差分析
7. 07 线性回归与 softmax 回归（理解与应用）
8. 08 正则化与偏差-方差权衡
