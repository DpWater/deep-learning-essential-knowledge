# 数学手册（线性神经网络进阶版）

定位：这份手册按“高于 D2L 线性网络章节（3.1~3.7）”设计。目标不是只会背公式，而是能推导、能实现、能排错、能解释为什么有效。

## 0. 使用方式
- 你想快速开工：先看第 1 节、第 5 节、第 6 节。
- 你想搞懂原理：重点看第 2 节（线性回归）和第 3 节（softmax 回归）。
- 你想查某个概念：每个词条都固定 6 项：`含义`、`定理/公式`、`图像意义`、`物理意义`、`深度学习运用`、`拿来就用`。

记号约定：
- `X in R^{n x d}`：`n` 条样本，`d` 维特征
- `y in R^n`（回归）或 `Y in R^{n x q}`（分类）
- `W in R^{d x q}`，`b in R^q`
- `z = XW + b`，`p = softmax(z)`
- `L` 表示经验风险（平均损失）

## 1. 先修关系与能力目标

### 1.1 先修关系
- 线性代数：向量、矩阵乘法、转置、逆、特征值。
- 微积分：链式法则、梯度、Hessian。
- 概率：高斯分布、最大似然、交叉熵。

### 1.2 学完你应具备的能力
- 能从概率假设推导出线性回归和 softmax 的损失函数。
- 能手推 `dL/dW`、`dL/db`、`dL/dz`（含 `p - y`）。
- 能解释为什么要 `logsumexp`、为什么 `CrossEntropyLoss` 直接吃 logits。
- 能用诊断流程解决“loss 不降、nan、过拟合、分类塌陷”等常见问题。

## 2. 线性回归（从统计假设到优化）

### 2.1 词条 L1：仿射线性模型
- 含义：输出由输入线性组合再加常数偏置。
- 定理/公式：`y_hat = w^T x + b`，批量形式 `y_hat = Xw + b`。
- 图像意义：2D 是直线，3D 是平面，高维是超平面。
- 物理意义：线性系统的静态响应，`b` 是系统零点漂移。
- 深度学习运用：线性层、分类头、回归头、线性 probe。
- 拿来就用：任何监督任务先用线性头做 baseline，评估特征是否可分。

### 2.2 词条 L2：平方损失（MSE）
- 含义：惩罚预测误差的平方，较大误差惩罚更重。
- 定理/公式：`L = (1/2n) ||Xw + b - y||_2^2`。
- 图像意义：损失面是抛物碗，单峰。
- 物理意义：最小能量拟合，偏离越大势能越高。
- 深度学习运用：回归任务标准损失，重建误差常用。
- 拿来就用：如果标签噪声近似高斯，优先用 MSE。

### 2.3 词条 L3：高斯噪声与 MLE 等价
- 含义：若 `y = w^T x + b + eps` 且 `eps ~ N(0, sigma^2)`，MLE 等价最小化 MSE。
- 定理/公式：
  - `p(y|x) = N(w^T x + b, sigma^2)`
  - `argmax log p(D|w,b) <=> argmin ||Xw+b-y||^2`
- 图像意义：点围绕回归面做高斯散布。
- 物理意义：测量误差是许多小扰动叠加（CLT）。
- 深度学习运用：概率解释损失函数选择，不再“拍脑袋”。
- 拿来就用：先判断噪声模型再选损失；重尾噪声改用 MAE/Huber。

### 2.4 词条 L4：正规方程（解析解）
- 含义：线性回归在二次损失下可直接闭式求解。
- 定理/公式：`w* = (X^T X)^(-1) X^T y`（含偏置时给 `X` 加全 1 列）。
- 图像意义：一次性跳到抛物碗最低点。
- 物理意义：最小二乘正交投影。
- 深度学习运用：小规模问题可作“真值解”校验 SGD 实现。
- 拿来就用：当 `d` 不大时，先求解析解作为训练结果上界检查。

### 2.5 词条 L5：凸性与 Hessian
- 含义：损失是凸函数，极小值唯一（在满秩条件下）。
- 定理/公式：
  - `L(w) = (1/2n)||Xw-y||^2`
  - `grad = (1/n) X^T(Xw-y)`
  - `H = (1/n) X^T X`（半正定）
- 图像意义：碗形地形无局部陷阱。
- 物理意义：正刚度系统的稳定平衡点。
- 深度学习运用：解释线性模型为何容易训练，便于对比深网非凸性。
- 拿来就用：若优化很慢，检查条件数 `kappa(X^T X)`，必要时标准化或加正则。

### 2.6 词条 L6：SGD 在线性回归中的行为
- 含义：用小批量梯度近似全量梯度，带噪但高效。
- 定理/公式：`w <- w - eta * (1/|B|) sum_{i in B} x_i (w^T x_i + b - y_i)`。
- 图像意义：沿着谷底前进并有小幅随机抖动。
- 物理意义：受随机扰动的阻尼系统。
- 深度学习运用：大数据下解析解不可行时的默认方法。
- 拿来就用：`batch` 太小会抖，太大更新慢；从 `32/64/128` 起步。

### 2.7 词条 L7：Ridge/Lasso 与 MAP
- 含义：正则化是“数据拟合 + 参数先验”的折中。
- 定理/公式：
  - Ridge: `L + lambda ||w||_2^2`（高斯先验）
  - Lasso: `L + lambda ||w||_1`（拉普拉斯先验）
- 图像意义：在损失碗上加“参数拉回力”。
- 物理意义：阻止系统振幅无限增大。
- 深度学习运用：weight decay、稀疏化、提升泛化。
- 拿来就用：先上 Ridge（更稳定），再按需求尝试 Lasso。

### 2.8 词条 L8：偏差-方差权衡
- 含义：模型太简单偏差高，太复杂方差高。
- 定理/公式：`generalization error = bias^2 + variance + noise`。
- 图像意义：训练误差和验证误差形成 U 形关系。
- 物理意义：欠拟合是表达能力不足，过拟合是噪声放大。
- 深度学习运用：决定是否增大模型、加强正则、补数据。
- 拿来就用：只看训练集结论常错，必须同时看验证集曲线。

### 2.9 词条 L9：评价指标（回归）
- 含义：不同指标衡量误差侧重点不同。
- 定理/公式：
  - `MSE, RMSE, MAE, R^2`
- 图像意义：RMSE 强调大误差，MAE 更稳健。
- 物理意义：不同代价函数对应不同工程风险偏好。
- 深度学习运用：业务口径决定指标，不是只看一个 loss。
- 拿来就用：报告至少给 `MAE + RMSE + R^2` 三个指标。

### 2.10 词条 L10：线性回归在深度学习中的现实位置
- 含义：线性不是“过时模型”，而是深网读出层和诊断工具。
- 定理/公式：`head(h) = W h + b`。
- 图像意义：深特征空间里的线性切分面。
- 物理意义：复杂动力学后的线性读出器。
- 深度学习运用：
  - 预训练模型线性 probe
  - 冻结 backbone 只训练 head
  - 蒸馏中的线性投影层
- 拿来就用：迁移学习先冻住 backbone，训练线性头快速验证数据可学性。

## 3. Softmax 回归（多分类核心）

### 3.1 词条 S1：logit 与类别概率
- 含义：logit 是未归一化得分，softmax 把得分变成概率。
- 定理/公式：
  - `z = XW + b`
  - `p_j = exp(z_j) / sum_k exp(z_k)`
- 图像意义：每个类别对应一张线性评分面，softmax 做归一化竞争。
- 物理意义：玻尔兹曼分布式的能量竞争（低能更可能）。
- 深度学习运用：几乎所有多分类头。
- 拿来就用：推理时 `argmax(logits)` 与 `argmax(softmax(logits))` 一致。

### 3.2 词条 S2：交叉熵与最大似然
- 含义：最小交叉熵等价于最大化标签条件概率。
- 定理/公式：
  - `L = -(1/n) sum_i sum_j y_ij log p_ij`
  - `argmin CE <=> argmax log p(Y|X)`
- 图像意义：把预测分布 `p` 拉向真实分布 `y`。
- 物理意义：编码代价最小化。
- 深度学习运用：分类训练标准目标。
- 拿来就用：多分类默认从 CE 开始，不要先自定义奇怪损失。

### 3.3 词条 S3：关键梯度 `p - y`
- 含义：误差梯度就是“预测概率 - 真实分布”。
- 定理/公式：`dL/dz = p - y`。
- 图像意义：预测过高的类被压，预测过低的类被拉。
- 物理意义：误差反馈力正比于概率差。
- 深度学习运用：反向传播高效稳定的核心原因。
- 拿来就用：梯度检查时先验证这条式子，能排掉 80% 实现错误。

### 3.4 词条 S4：softmax 的平移不变性
- 含义：logits 同时加一个常数，softmax 概率不变。
- 定理/公式：`softmax(z) = softmax(z - c)`。
- 图像意义：整体抬高/压低评分面不改变相对排序。
- 物理意义：改变能量零点不改变相对概率。
- 深度学习运用：`logsumexp` 稳定实现的理论基础。
- 拿来就用：实现 softmax 时先减 `max(z)`。

### 3.5 词条 S5：数值稳定（logsumexp）
- 含义：避免 `exp(large)` 溢出与 `exp(small)` 下溢。
- 定理/公式：`log sum exp(z) = m + log sum exp(z-m)`，`m=max(z)`。
- 图像意义：把曲线横向平移到安全区计算。
- 物理意义：改参考点但不改相对能级差。
- 深度学习运用：CE、CRF、能量模型全都依赖它。
- 拿来就用：在 PyTorch 用 `F.cross_entropy(logits, y)`，别手搓 softmax+log。

### 3.6 词条 S6：softmax 回归的凸性
- 含义：对参数是凸优化（线性模型 + log-sum-exp）。
- 定理/公式：
  - 单样本损失 `l(z)=log sum exp(z)-y^T z`
  - `log sum exp` 为凸函数
- 图像意义：单盆地，无坏局部极小。
- 物理意义：稳定势能面。
- 深度学习运用：作为“理论清晰”的分类基线。
- 拿来就用：复杂模型跑不稳时，先用 softmax 线性头验证数据标签质量。

### 3.7 词条 S7：类别不平衡
- 含义：多数类压制少数类，整体 accuracy 可能虚高。
- 定理/公式：
  - 加权 CE：`L = -sum_j alpha_j y_j log p_j`
  - focal loss：`-(1-p_t)^gamma log p_t`
- 图像意义：提高少数类样本在损失中的“音量”。
- 物理意义：对稀有事件施加更高风险权重。
- 深度学习运用：医疗、风控、缺陷检测常见。
- 拿来就用：先试 class weight，再看 PR-AUC / F1，而非只看 accuracy。

### 3.8 词条 S8：标签平滑（Label Smoothing）
- 含义：把 one-hot 稍微“软化”，防止过度自信。
- 定理/公式：`y_smooth = (1-eps)*y + eps/q`。
- 图像意义：把概率目标从尖峰改为窄峰。
- 物理意义：承认标注和观测存在不确定性。
- 深度学习运用：提升泛化和校准，常用于大分类任务。
- 拿来就用：从 `eps=0.05~0.1` 尝试。

### 3.9 词条 S9：温度缩放与校准
- 含义：模型可能分类对但概率不准，温度缩放用于校准。
- 定理/公式：`p = softmax(z/T)`，`T>1` 更平滑，`T<1` 更尖锐。
- 图像意义：同一排名下，概率曲线变陡或变平。
- 物理意义：有效“热噪声”调节系统确定性。
- 深度学习运用：后处理概率输出，提升可靠性。
- 拿来就用：在验证集单独拟合 `T`，不改主模型参数。

### 3.10 词条 S10：线性分类头在大模型中的作用
- 含义：即使 backbone 很深，最终决策常由线性头完成。
- 定理/公式：`logits = H W + b`。
- 图像意义：深特征空间里的超平面分割。
- 物理意义：复杂特征提取后由简单判别器读出。
- 深度学习运用：微调、检索、蒸馏、对齐任务。
- 拿来就用：新任务先只训线性头，确认数据与特征是否匹配。

## 4. 超过基础章节的关键补充（超纲但高频）

### 4.1 二阶信息与预条件化
- 含义：不同参数方向曲率不同，统一学习率不总是合适。
- 定理/公式：二次近似 `L(theta+delta) ~= L + g^T delta + 1/2 delta^T H delta`。
- 图像意义：椭圆谷底需要各向异性步长。
- 物理意义：不同方向等效刚度不同。
- 深度学习运用：Adam/RMSProp 本质是对角预条件。
- 拿来就用：loss 震荡时先降 lr，其次考虑优化器与归一化策略。

### 4.2 Fisher 信息与自然梯度（概念级）
- 含义：在概率模型空间中，参数变化应按分布几何度量。
- 定理/公式：`delta = -eta * F^{-1} g`（自然梯度形式）。
- 图像意义：在“分布流形”上走最陡下降，而非欧氏坐标最陡。
- 物理意义：按真实可观测变化做最省力更新。
- 深度学习运用：二阶近似法、K-FAC 思想来源。
- 拿来就用：先理解思想，不必立刻上复杂实现。

### 4.3 分布偏移下的线性头策略
- 含义：训练分布与线上分布不一致会导致性能掉点。
- 定理/公式：关注 `p_train(x,y)` 与 `p_test(x,y)` 差异。
- 图像意义：决策边界在新分布上偏移。
- 物理意义：系统工作点改变，原参数不再最优。
- 深度学习运用：域适配、重加权、阈值重设。
- 拿来就用：上线前做校准与阈值重估，不要直接照搬训练阈值。

## 5. 可直接复用的训练模板（PyTorch）

### 5.1 线性回归最小模板
```python
import torch
import torch.nn as nn
import torch.optim as optim

model = nn.Linear(d_in, 1)
criterion = nn.MSELoss()
optimizer = optim.SGD(model.parameters(), lr=1e-2, momentum=0.9)

for x, y in loader:
    pred = model(x).squeeze(-1)
    loss = criterion(pred, y)
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

### 5.2 多分类 softmax 回归最小模板
```python
import torch
import torch.nn as nn
import torch.optim as optim

model = nn.Linear(d_in, num_classes)
criterion = nn.CrossEntropyLoss(label_smoothing=0.05)
optimizer = optim.AdamW(model.parameters(), lr=3e-4, weight_decay=1e-2)

for x, y in loader:
    logits = model(x)              # [batch, num_classes]
    loss = criterion(logits, y)    # y: int64 class index
    optimizer.zero_grad()
    loss.backward()
    torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
    optimizer.step()
```

### 5.3 形状与类型硬规则
- 回归：`pred` 和 `y` 必须同形。
- 多分类 CE：`logits` 形状 `[B, C]`，`y` 形状 `[B]` 且 `dtype=torch.long`。
- 二分类：优先 `BCEWithLogitsLoss`，输入是 logits，不是 sigmoid 后概率。

## 6. 诊断与排错字典（高频故障）

### 故障 D1：`loss = nan`
- 常见原因：学习率过大、输入含 nan/inf、手写 softmax 溢出。
- 快速排查：
  - 打印 `x.isnan().any()`、`torch.isinf(x).any()`
  - 把 lr 降 10 倍
  - 改用框架自带 `CrossEntropyLoss`

### 故障 D2：训练集降、验证集不降
- 常见原因：过拟合、数据泄漏、验证集分布不一致。
- 快速排查：
  - 加 weight decay / label smoothing
  - 检查 train/val 是否重复样本
  - 做分层抽样

### 故障 D3：分类只预测一个类
- 常见原因：类别极不平衡、标签编码错、最后一层维度错。
- 快速排查：
  - 统计每类样本数
  - 检查 `y` 是否在 `[0, C-1]`
  - 加 class weight

### 故障 D4：梯度一直很小
- 常见原因：输入尺度过小、初始化不当、归一化缺失。
- 快速排查：
  - 打印 `grad_norm`
  - 做输入标准化
  - 调整初始化和优化器

## 7. 一页速用（开训前 90 秒）
- 任务类型是否明确：回归 / 二分类 / 多分类。
- 输出层是否匹配：`Linear(d_in,1)` 或 `Linear(d_in,C)`。
- 损失是否匹配：MSE / BCEWithLogits / CrossEntropy。
- 标签格式是否匹配：
  - CE 要类索引，不是 one-hot。
  - BCE 的标签要 float。
- 学习率是否在合理区间。
- 是否开启基础正则：`weight_decay`、可选 `label_smoothing`。
- 是否监控：train loss、val loss、主指标（MAE/F1/AUC）。

## 8. 练习（附思路）

### 练习 1：推导线性回归解析解
- 题目：对 `L(w)=||Xw-y||^2` 求最优 `w`。
- 思路：
  - 对 `w` 求梯度并令零：`2X^T(Xw-y)=0`
  - 得 `X^TXw=X^Ty`
  - 若 `X^TX` 可逆，`w=(X^TX)^(-1)X^Ty`

### 练习 2：证明 `dL/dz = p - y`
- 题目：softmax + CE 推导。
- 思路：
  - 把 CE 写成 `-sum y_j log softmax(z)_j`
  - 展开成 `logsumexp(z)-y^T z`
  - 对 `z` 求导即可得到 `softmax(z)-y`

### 练习 3：为什么 softmax 要减最大值
- 题目：证明平移不变性。
- 思路：
  - 将 `z_j` 全部替换为 `z_j-c`
  - 分子分母都乘 `exp(-c)`，比值不变
  - 取 `c=max(z)` 可消除溢出风险

## 9. 学习路径（从这份手册继续往上）
- 第 1 阶段：线性回归 + softmax 完整手推一遍。
- 第 2 阶段：从零实现（不依赖高阶 API）并做数值梯度检查。
- 第 3 阶段：加入正则、类不平衡、校准、阈值优化。
- 第 4 阶段：迁移到 MLP/Transformer 时，把“线性头诊断法”保留。
