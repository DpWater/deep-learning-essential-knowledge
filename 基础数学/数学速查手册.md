# 数学速查手册（字典版，深度学习可直接套用）

用途：作为“数学字典”直接查概念、定理、图像意义、物理意义和深度学习用法。

## 0. 使用说明
- 每个词条固定包含：`含义`、`核心定理/公式`、`图像意义`、`物理意义`、`深度学习中的涉及与运用`、`拿来就用`。
- 先看“按任务速查”，再跳到对应词条。
- 记号：样本矩阵 `X∈R^{n×d}`，参数矩阵 `W`，损失 `L`，预测 `p`。

## 1. 线性代数词条

### 词条 1：向量内积（Dot Product）
- 含义：两个向量在方向上的一致程度与长度的乘积。
- 核心定理/公式：`x^T y = ||x||_2 ||y||_2 cos(theta)`。
- 图像意义：一个向量在另一个向量上的“投影长度乘以该向量长度”。
- 物理意义：功 `W = F·s`，力在位移方向上的有效分量。
- 深度学习中的涉及与运用：注意力打分 `q·k`、相似度检索、余弦相似度。
- 拿来就用：做 embedding 检索时，用“向量归一化 + 内积”近似余弦相似度。

### 词条 2：矩阵乘法（Matrix Multiplication）
- 含义：把一个线性变换作用到向量/矩阵上。
- 核心定理/公式：`(AB)_{ij}=sum_k A_{ik}B_{kj}`。
- 图像意义：旋转、缩放、剪切等线性几何变换的组合。
- 物理意义：多阶段线性系统串联后的整体映射。
- 深度学习中的涉及与运用：全连接层 `Y=XW+b`、QKV 投影、卷积的 im2col 形式。
- 拿来就用：先检查 shape：`[batch,d_in] x [d_in,d_out] -> [batch,d_out]`。

### 词条 3：特征值与特征向量（Eigen）
- 含义：在线性变换下方向不变、只缩放的特殊方向。
- 核心定理/公式：`Av=lambda v`。
- 图像意义：椭球主轴方向，对应“拉伸最强/最弱”的方向。
- 物理意义：振动系统的固有模态与固有频率。
- 深度学习中的涉及与运用：Hessian 曲率分析、谱归一化、稳定性分析。
- 拿来就用：如果训练不稳，先看最大特征值（或谱半径）是否过大。

### 词条 4：奇异值分解（SVD）
- 含义：任意矩阵分解为“旋转 + 缩放 + 旋转”。
- 核心定理/公式：`A=U Sigma V^T`，奇异值非负。
- 图像意义：单位球经变换后成为椭球，半轴长度就是奇异值。
- 物理意义：系统各正交通道的能量增益。
- 深度学习中的涉及与运用：低秩压缩、LoRA 思路、权重谱分析。
- 拿来就用：模型太大时，先试“截断 SVD”压缩线性层。

### 词条 5：正定矩阵与二次型（Positive Definite）
- 含义：对任意非零向量都有正能量 `x^T A x > 0`。
- 核心定理/公式：对称正定矩阵全部特征值 `>0`。
- 图像意义：二次型等高线是同心椭圆/椭球。
- 物理意义：弹簧系统势能一定非负且有唯一平衡点。
- 深度学习中的涉及与运用：二阶优化近似、协方差矩阵、马氏距离。
- 拿来就用：做二次近似时，确保 Hessian 近似正定（必要时加 `lambda I`）。

### 词条 6：范数（Norm）
- 含义：向量/矩阵大小的度量。
- 核心定理/公式：`L1=sum |x_i|`，`L2=sqrt(sum x_i^2)`，`Frobenius=sqrt(sum A_ij^2)`。
- 图像意义：L2 单位球是圆，L1 单位球是菱形（更容易产生稀疏角点）。
- 物理意义：L2 对应能量，L1 对应“总绝对消耗”。
- 深度学习中的涉及与运用：L2 权重衰减、L1 稀疏化、梯度范数监控。
- 拿来就用：过拟合先加 `weight_decay`（L2），要稀疏特征再考虑 L1。

## 2. 微积分与优化词条

### 词条 7：梯度（Gradient）
- 含义：函数在当前位置增长最快方向。
- 核心定理/公式：`grad f(x) = [partial f / partial x_i]`。
- 图像意义：等高线图中垂直于等高线并指向最陡上升。
- 物理意义：势场中的“受力方向”。
- 深度学习中的涉及与运用：反向传播得到参数梯度并更新。
- 拿来就用：最小化损失用负梯度方向更新参数。

### 词条 8：Jacobian（雅可比矩阵）
- 含义：向量到向量映射的一阶导数总表。
- 核心定理/公式：`J_ij = partial y_i / partial x_j`。
- 图像意义：局部线性化变换矩阵。
- 物理意义：多变量系统的小扰动传递增益。
- 深度学习中的涉及与运用：normalizing flow 的行列式项、敏感度分析。
- 拿来就用：要看输入扰动如何放大到输出，直接分析 Jacobian 谱范数。

### 词条 9：Hessian（海森矩阵）
- 含义：标量函数的二阶导数矩阵，描述曲率。
- 核心定理/公式：`H_ij = partial^2 f / partial x_i partial x_j`。
- 图像意义：碗口陡峭程度与主曲率方向。
- 物理意义：局部“刚度矩阵”。
- 深度学习中的涉及与运用：二阶方法、sharp/flat minima 分析。
- 拿来就用：学习率调不稳时，检查曲率大方向，必要时降 lr 或做预条件化。

### 词条 10：链式法则（Chain Rule）
- 含义：复合函数求导法则。
- 核心定理/公式：`dL/dx = dL/du * du/dx`；多层网络是连乘。
- 图像意义：局部斜率逐层传递与缩放。
- 物理意义：级联系统增益相乘。
- 深度学习中的涉及与运用：反向传播的数学基础。
- 拿来就用：调试梯度时按“从 loss 到输入”逐层检查导数链路。

### 词条 11：泰勒展开（Taylor Expansion）
- 含义：用多项式近似函数的局部行为。
- 核心定理/公式：`f(x+dx) ~= f(x) + g^T dx + 1/2 dx^T H dx`。
- 图像意义：一阶是切线，二阶是抛物面。
- 物理意义：平衡点附近的近似动力学。
- 深度学习中的涉及与运用：牛顿法、损失局部近似、收敛分析。
- 拿来就用：做小步更新假设时，先看一阶项，再用二阶项估计步长风险。

### 词条 12：凸性与 Jensen 不等式
- 含义：凸函数满足“连线在函数图像上方”。
- 核心定理/公式：Jensen：`phi(E[X]) <= E[phi(X)]`（`phi` 凸）。
- 图像意义：弓形曲线下，先平均再过函数更小。
- 物理意义：混合态的代价不低于平均代价。
- 深度学习中的涉及与运用：ELBO 推导、损失上界构造、泛化分析。
- 拿来就用：看到期望里有非线性函数，优先尝试 Jensen 做上下界。

### 词条 13：Lipschitz 连续
- 含义：函数变化率有全局上界。
- 核心定理/公式：`||f(x)-f(y)|| <= L ||x-y||`。
- 图像意义：函数斜率不会无限大。
- 物理意义：系统输出对输入扰动的最大放大倍数受限。
- 深度学习中的涉及与运用：WGAN 梯度惩罚、鲁棒性、步长上界。
- 拿来就用：训练 GAN 不稳时，加入梯度惩罚以近似 1-Lipschitz。

## 3. 概率统计词条

### 词条 14：条件概率与贝叶斯公式
- 含义：在已知条件下更新事件概率。
- 核心定理/公式：`P(A|B)=P(B|A)P(A)/P(B)`。
- 图像意义：样本空间在条件子集上的“重新归一化”。
- 物理意义：观测到证据后对系统状态做后验修正。
- 深度学习中的涉及与运用：朴素贝叶斯、后验推断、生成模型。
- 拿来就用：类别先验变化时，用贝叶斯后验校正输出概率。

### 词条 15：期望、方差、协方差
- 含义：均值反映中心，方差反映离散，协方差反映联动。
- 核心定理/公式：`Var(X)=E[X^2]-E[X]^2`，`Cov(X,Y)=E[(X-EX)(Y-EY)]`。
- 图像意义：分布中心与散布椭圆方向。
- 物理意义：噪声强度与变量耦合。
- 深度学习中的涉及与运用：BatchNorm、特征标准化、不确定性建模。
- 拿来就用：输入特征先做标准化，通常能显著加快收敛。

### 词条 16：大数定律（LLN）
- 含义：样本均值收敛到总体期望。
- 核心定理/公式：`(1/n) sum X_i -> E[X]`（`n->infinity`）。
- 图像意义：均值曲线随样本增多趋于稳定。
- 物理意义：重复实验抹平随机波动。
- 深度学习中的涉及与运用：mini-batch 梯度是总体梯度估计。
- 拿来就用：batch 太小噪声大，适度增大 batch 或做梯度累积。

### 词条 17：中心极限定理（CLT）
- 含义：大量独立同分布变量和趋近高斯。
- 核心定理/公式：标准化和的分布趋向 `N(0,1)`。
- 图像意义：很多小扰动叠加后形成钟形曲线。
- 物理意义：宏观噪声常近似高斯。
- 深度学习中的涉及与运用：初始化分布、噪声建模、置信区间近似。
- 拿来就用：缺少精确噪声模型时，先用高斯噪声作为一阶近似。

### 词条 18：最大似然估计（MLE）
- 含义：找使观测数据最可能出现的参数。
- 核心定理/公式：`theta*=argmax sum_i log p(x_i|theta)`。
- 图像意义：在参数空间找似然峰值。
- 物理意义：寻找最能解释实验数据的系统参数。
- 深度学习中的涉及与运用：回归、分类、语言模型训练本质都是 MLE。
- 拿来就用：先把问题写成概率模型，再对数似然求和最小化负号。

### 词条 19：MAP 估计
- 含义：在 MLE 上加入先验偏好。
- 核心定理/公式：`theta*=argmax [log p(D|theta)+log p(theta)]`。
- 图像意义：在似然地形上叠加先验“势场”。
- 物理意义：数据证据与先验知识的平衡。
- 深度学习中的涉及与运用：L2 正则相当于高斯先验下的 MAP。
- 拿来就用：数据少易过拟合时，用先验（正则）稳住参数。

## 4. 信息论与损失函数词条

### 词条 20：熵（Entropy）
- 含义：分布不确定性的度量。
- 核心定理/公式：`H(P)=-sum_i p_i log p_i`。
- 图像意义：分布越平坦，熵越高。
- 物理意义：系统无序度/信息缺失程度。
- 深度学习中的涉及与运用：标签平滑、主动学习、不确定性评估。
- 拿来就用：分类过度自信时，可用标签平滑提升泛化。

### 词条 21：交叉熵（Cross-Entropy）
- 含义：用模型分布 `Q` 编码真实分布 `P` 的平均代价。
- 核心定理/公式：`H(P,Q)=-sum_i p_i log q_i`。
- 图像意义：`Q` 偏离 `P` 越大，代价越高。
- 物理意义：编码系统与真实信源失配带来的额外码长。
- 深度学习中的涉及与运用：分类任务标准损失。
- 拿来就用：多分类直接用 `CrossEntropyLoss(logits, y)`。

### 词条 22：KL 散度（Kullback-Leibler）
- 含义：两个分布的相对差异（非对称）。
- 核心定理/公式：`KL(P||Q)=sum_i p_i log(p_i/q_i)`，且 `>=0`。
- 图像意义：`Q` 用来拟合 `P` 时的信息损失。
- 物理意义：用错误模型解释真实系统的额外代价。
- 深度学习中的涉及与运用：VAE、蒸馏、分布对齐。
- 拿来就用：蒸馏时用温度 softmax 后做 KL 对齐 teacher/student。

### 词条 23：Softmax + Cross-Entropy 梯度
- 含义：多分类中最关键的可导组合。
- 核心定理/公式：若 `y` 为 one-hot，`dL/dz = p - y`。
- 图像意义：预测概率向真实类别方向“拉近”。
- 物理意义：误差力场推动 logits 修正。
- 深度学习中的涉及与运用：所有多分类头（NLP/CV/推荐）。
- 拿来就用：不要手写 softmax 再 CE，直接用框架融合版本防数值溢出。

### 词条 24：MSE 与 MAE
- 含义：回归误差的二次惩罚与一次惩罚。
- 核心定理/公式：`MSE=mean((y-yhat)^2)`，`MAE=mean(|y-yhat|)`。
- 图像意义：MSE 抛物线、MAE 折线。
- 物理意义：MSE 对大误差敏感，MAE 更抗离群点。
- 深度学习中的涉及与运用：数值预测、重建任务。
- 拿来就用：噪声近高斯选 MSE；离群点多选 MAE/Huber。

## 5. 优化算法词条

### 词条 25：梯度下降（GD/SGD）
- 含义：沿负梯度方向迭代降低损失。
- 核心定理/公式：`theta_{t+1}=theta_t-lr * g_t`。
- 图像意义：在损失地形上“下山”。
- 物理意义：阻尼系统向低势能演化。
- 深度学习中的涉及与运用：全部参数训练的基础。
- 拿来就用：先用 Adam 跑通，再切 SGD+momentum 微调泛化。

### 词条 26：动量法（Momentum）
- 含义：对历史梯度做指数平均，减少来回震荡。
- 核心定理/公式：`v_t=beta v_{t-1}+(1-beta)g_t`，`theta-=lr*v_t`。
- 图像意义：峡谷地形中沿主方向加速、横向减摆。
- 物理意义：带惯性的粒子受阻尼前进。
- 深度学习中的涉及与运用：CV 大模型常用 SGD+momentum。
- 拿来就用：默认 `momentum=0.9` 通常是安全起点。

### 词条 27：Adam
- 含义：一阶矩（均值）+二阶矩（方差）自适应步长。
- 核心定理/公式：`m,v` 指数滑动平均 + 偏差修正后更新。
- 图像意义：不同参数坐标轴自动缩放步长。
- 物理意义：在噪声环境中用自适应阻尼控制位移。
- 深度学习中的涉及与运用：Transformer、扩散模型常用优化器。
- 拿来就用：默认 `lr=1e-3`（小模型）或 `1e-4~5e-4`（大模型微调）。

### 词条 28：学习率与收敛稳定性
- 含义：每次参数更新的步幅。
- 核心定理/公式：二次近似下步长过大可能发散。
- 图像意义：步子太大越过谷底，步子太小收敛极慢。
- 物理意义：积分步长决定系统数值稳定性。
- 深度学习中的涉及与运用：warmup、cosine decay、one-cycle。
- 拿来就用：训练前 3 个 epoch 开 warmup，后续 cosine 衰减。

### 词条 29：梯度裁剪（Gradient Clipping）
- 含义：限制梯度向量范数避免爆炸。
- 核心定理/公式：若 `||g||>c`，用 `g <- c * g/||g||`。
- 图像意义：把过长更新箭头截断到球面内。
- 物理意义：给系统施加最大推力约束。
- 深度学习中的涉及与运用：RNN、长序列 Transformer、混合精度训练。
- 拿来就用：先试 `clip_grad_norm_=1.0`。

## 6. 数值稳定与分布词条

### 词条 30：LogSumExp 技巧
- 含义：稳定计算 `log(sum(exp(x)))`。
- 核心定理/公式：`m=max(x)`，`LSE(x)=m+log(sum(exp(x-m)))`。
- 图像意义：整体平移后再指数，避免极值溢出。
- 物理意义：把能量参考点平移，物理量差值不变。
- 深度学习中的涉及与运用：softmax、CRF、能量模型。
- 拿来就用：凡是出现 `log(sum(exp()))`，都替换成 `logsumexp`。

### 词条 31：高斯分布（Gaussian）
- 含义：由均值与方差刻画的连续分布。
- 核心定理/公式：`N(mu,sigma^2)`，密度与二次型指数相关。
- 图像意义：钟形曲线，`mu` 决定中心，`sigma` 决定宽度。
- 物理意义：大量独立小扰动叠加的自然结果。
- 深度学习中的涉及与运用：初始化、噪声建模、VAE 隐变量先验。
- 拿来就用：无先验时，回归残差常先假设高斯噪声。

### 词条 32：重参数化技巧（Reparameterization）
- 含义：把随机采样写成“可导变换 + 固定噪声”。
- 核心定理/公式：`z = mu + sigma * eps`，`eps~N(0,I)`。
- 图像意义：先在标准噪声空间采样，再仿射映射到目标分布。
- 物理意义：随机性外置，系统参数只控制变换。
- 深度学习中的涉及与运用：VAE、随机隐变量模型。
- 拿来就用：需要对采样路径反传梯度时，优先考虑重参数化。

### 词条 33：BatchNorm / LayerNorm 的数学本质
- 含义：对激活做标准化并学习可恢复的缩放平移。
- 核心定理/公式：`xhat=(x-mu)/sqrt(var+eps)`，`y=gamma*xhat+beta`。
- 图像意义：把特征拉回统一尺度后再平移缩放。
- 物理意义：控制系统状态量的量纲与幅值，降低病态条件数。
- 深度学习中的涉及与运用：加速训练、缓解梯度问题。
- 拿来就用：CNN 常用 BN，Transformer 常用 LN/RMSNorm。

## 8. 训练前数学检查清单
- `shape` 是否一致：`X @ W + b` 的维度严格对齐。
- `loss` 与任务是否匹配：分类用 CE，回归用 MSE/MAE。
- `数值稳定` 是否处理：CE 用 logits 接口、使用 logsumexp。
- `梯度` 是否健康：监控梯度范数，必要时裁剪。
- `学习率` 是否合理：先 warmup，再衰减。
- `归一化` 是否就位：输入标准化 + 合适的 Norm 层。
