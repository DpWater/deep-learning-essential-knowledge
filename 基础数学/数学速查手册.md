## 1. 符号与维度约定
- 标量：`a ∈ R`
- 向量：`x ∈ R^d`，默认列向量
- 矩阵：`X ∈ R^{n×d}`（`n` 条样本，`d` 维特征）
- 模型参数：`W ∈ R^{d_in×d_out}`，`b ∈ R^{d_out}`
- 预测：`z = XW + b`，`z ∈ R^{n×d_out}`
- 梯度：`∂L/∂x` 与 `x` 同形状

速记：
- 前向看 shape，反向看同形。
- 批量维一般放第 0 维：`[batch, ...]`。

## 2. 线性代数速查
### 2.1 常用运算
- 点积：`x^T y = Σ_i x_i y_i`，衡量方向相似度。
- 矩阵乘法：`(AB)_{ij} = Σ_k A_{ik}B_{kj}`。
- 转置：`(AB)^T = B^T A^T`。
- 迹：`tr(A) = Σ_i A_{ii}`，且 `tr(AB) = tr(BA)`。
- 范数：
  - `||x||_1 = Σ_i |x_i|`
  - `||x||_2 = sqrt(Σ_i x_i^2)`
  - `||A||_F = sqrt(Σ_{ij} A_{ij}^2)`

### 2.2 常用矩阵性质
- 对称矩阵：`A = A^T`
- 正定矩阵：`x^T A x > 0`
- 正交矩阵：`Q^TQ = I`，逆等于转置

### 2.3 特征值与 SVD
- 特征分解：`Av = λv`（只适用于方阵）
- SVD：`A = UΣV^T`（任意矩阵）
- 降维/压缩常用：取最大奇异值对应的前 `k` 个分量

### 2.4 深度学习最常用结论
- 线性层：`y = xW + b`
- 协方差矩阵：`Cov(X) = E[(X-μ)(X-μ)^T]`
- PCA 本质：对协方差矩阵做特征分解

## 3. 微积分与矩阵求导
### 3.1 一元/多元基础
- 链式法则：`dy/dx = (dy/du)(du/dx)`
- 多变量链式法则：`∂L/∂x = (∂L/∂y)(∂y/∂x)`

### 3.2 高频求导公式
- `∂(a^T x)/∂x = a`
- `∂(x^T A x)/∂x = (A + A^T)x`
  - 若 `A` 对称：`= 2Ax`
- `∂||x||_2^2/∂x = 2x`
- `∂(Ax)/∂x = A`
- `∂(x^T x)/∂x = 2x`

### 3.3 激活函数导数
- `sigmoid(x) = 1/(1+e^{-x})`
  - `sigmoid'(x) = sigmoid(x)(1-sigmoid(x))`
- `tanh'(x) = 1 - tanh^2(x)`
- `ReLU'(x) = 1(x>0)`（`x=0` 处通常置 0）
- `LeakyReLU'(x) = 1(x>0) + α1(x<=0)`
- `GELU(x) ≈ 0.5x(1+tanh(sqrt(2/pi)(x+0.044715x^3)))`

### 3.4 Softmax 与交叉熵（必须会）
- `softmax(z_i) = exp(z_i)/Σ_j exp(z_j)`
- 稳定实现：`softmax(z) = exp(z-m)/Σ_j exp(z_j-m)`，`m=max(z)`
- 多分类交叉熵：`L = -Σ_i y_i log(p_i)`
- 若 `y` one-hot，梯度简化：`∂L/∂z = p - y`

## 4. 概率统计速查
### 4.1 概率基本式
- 条件概率：`P(A|B) = P(A,B)/P(B)`
- 贝叶斯公式：`P(A|B)=P(B|A)P(A)/P(B)`
- 全概率公式：`P(B)=Σ_i P(B|A_i)P(A_i)`

### 4.2 期望与方差
- 期望：`E[X] = Σ_x xP(x)` 或 `∫xp(x)dx`
- 方差：`Var(X)=E[(X-E[X])^2]=E[X^2]-E[X]^2`
- 协方差：`Cov(X,Y)=E[(X-E[X])(Y-E[Y])]`

### 4.3 常用统计量
- 标准差：`σ = sqrt(Var(X))`
- 相关系数：`ρ = Cov(X,Y)/(σ_Xσ_Y)`
- 样本均值：`x̄ = (1/n)Σ_i x_i`

### 4.4 最大似然与 MAP
- MLE：`θ* = argmax_θ Π_i p(x_i|θ)`
- 常转为 log-likelihood：`argmax_θ Σ_i log p(x_i|θ)`
- MAP：`argmax_θ log p(D|θ) + log p(θ)`（带先验）

## 5. 损失函数与信息论
### 5.1 回归
- MSE：`L = (1/n)Σ_i (y_i - ŷ_i)^2`
- MAE：`L = (1/n)Σ_i |y_i - ŷ_i|`
- Huber：
  - 小误差二次，大误差一次，抗离群点

### 5.2 分类
- 二分类 BCE：`L = -[y log p + (1-y)log(1-p)]`
- 多分类 CE：`L = -Σ_i y_i log p_i`

### 5.3 信息论
- 熵：`H(P) = -Σ_i p_i log p_i`
- 交叉熵：`H(P,Q) = -Σ_i p_i log q_i`
- KL 散度：`D_KL(P||Q)=Σ_i p_i log(p_i/q_i)`
- 关系：`H(P,Q)=H(P)+D_KL(P||Q)`

## 6. 优化算法公式
### 6.1 梯度下降家族
- GD：`θ_{t+1} = θ_t - η∇L(θ_t)`
- SGD：每个 mini-batch 更新一次

### 6.2 Momentum
- `v_t = βv_{t-1} + (1-β)g_t`
- `θ_{t+1} = θ_t - ηv_t`

### 6.3 RMSProp
- `s_t = ρs_{t-1} + (1-ρ)g_t^2`
- `θ_{t+1} = θ_t - η g_t/(sqrt(s_t)+ε)`

### 6.4 Adam
- `m_t = β1 m_{t-1} + (1-β1)g_t`
- `v_t = β2 v_{t-1} + (1-β2)g_t^2`
- 偏差修正：
  - `m̂_t = m_t/(1-β1^t)`
  - `v̂_t = v_t/(1-β2^t)`
- 更新：`θ_{t+1} = θ_t - η m̂_t/(sqrt(v̂_t)+ε)`

默认超参常用：
- `β1=0.9`, `β2=0.999`, `ε=1e-8`

## 7. 数值稳定与工程技巧
### 7.1 稳定计算
- `log(Σ_i exp(x_i))` 用 `logsumexp`：
  - `m=max(x)`，结果 `= m + log(Σ_i exp(x_i-m))`
- `softmax + cross_entropy` 不要分开手写，直接用框架融合实现
- 二分类用 `BCEWithLogitsLoss`，不要先 sigmoid 再 BCE

### 7.2 初始化
- Xavier（tanh/sigmoid 常用）：
  - `Var(W)=2/(fan_in+fan_out)`
- He（ReLU 家族常用）：
  - `Var(W)=2/fan_in`

### 7.3 正则化
- L2 正则：`L + λ||W||_2^2`
- L1 正则：`L + λ||W||_1`
- Dropout：训练时随机置零，推理时关闭

### 7.4 梯度问题排查
- 梯度爆炸：梯度裁剪 `clip_grad_norm_`
- 梯度消失：残差连接、归一化层、合适初始化
- 学不动：
  - 学习率过小/过大
  - 数据未标准化
  - loss 与输出层不匹配

## 8. 分布与采样
### 8.1 常见分布
- 伯努利：`X∈{0,1}`，参数 `p`
- 二项：`Bin(n,p)`
- 高斯：`N(μ,σ^2)`
- 类别分布：多分类标签建模

### 8.2 重参数化（VAE 核心）
- 若 `z ~ N(μ,σ^2)`，写为：
  - `z = μ + σ ⊙ ε`, `ε ~ N(0,I)`
- 作用：让采样步骤可反向传播

## 9. 常见排错字典
- 现象：`loss=nan`
  - 优先检查：学习率、除零、`log(0)`、输入是否含 `inf/nan`
- 现象：训练准确率高，验证差
  - 优先检查：过拟合、数据泄漏、增强策略
- 现象：输出全是同一类
  - 优先检查：类别不平衡、损失权重、最后一层与标签编码
- 现象：loss 不下降
  - 优先检查：标签错位、梯度是否为 0、优化器参数是否生效

## 10. 一页速用（训练前检查）
- `shape`：输入、标签、输出是否严格匹配
- `dtype`：特征 `float32`，分类标签通常 `int64`
- `loss`：任务与损失函数一致
- `lr`：先用常见默认值，小规模试跑
- `normalize`：输入做标准化/归一化
- `seed`：固定随机种子便于复现
